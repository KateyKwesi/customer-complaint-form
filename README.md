# Customer Complaint Form

A simple **Customer Complaint Form** built with **HTML, CSS, and JavaScript**.  
This project was **built to practice form validation, DOM manipulation, and handling checkboxes/radio buttons**.  
Includes **real-time validation**, **checkbox & radio group checks**, and **dynamic border feedback**.

---

## Features

- Real-time validation for all fields
- Regex validation for email, order number, and product code
- Checkbox and radio group validation
- Green border = valid, red border = invalid
- Full form validation on submit
- Helps practice JavaScript DOM manipulation and form handling

---

## Technologies

- HTML
- CSS
- JavaScript
